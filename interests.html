API verktyg

Det här var ett av de roligare projekten. Kort sagt så behövde min pappas jobb ett verktyg för att ladda upp filer till en databas med api. Jag tyckte det lät som en fantastisk möjlighet att lära sig mer om Python och api, så jag började direkt. Det var också första gången jag gjorde något med en användargränssnitt. Det slutgiltiga programmet kunde läsa in data från en excel fil, konvertera den till pdf filer, signera filerna och sedan skicka iväg dem.
